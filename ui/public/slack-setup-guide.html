Slack Setup Guide

How to create a Slack app and connect it to Adjutant so you can @mention the bot in any channel.

💬 Slack

Step 1 — Create a Slack App

  1. 1

    Go to api.slack.com/apps and click Create New App.

  2. 2

    Choose From scratch.

  3. 3

    Name your app (e.g. Adjutant) and select your workspace, then click Create App.

Step 2 — Enable Socket Mode

  1. 1

    In the left sidebar, click Socket Mode.

  2. 2

    Toggle Enable Socket Mode to ON.

  3. 3

    Click Generate to create an App-Level Token.

  4. 4

    Name it (e.g. adjutant-socket), add the connections:write scope, and click Generate.

  5. 5

    Copy the token — it starts with xapp-.

Step 3 — Subscribe to Events

  1. 1

    In the left sidebar, click Event Subscriptions.

  2. 2

    Toggle Enable Events to ON. (Socket Mode fills in the Request URL automatically — no public URL needed.)

  3. 3

    Expand Subscribe to bot events and click Add Bot User Event.

  4. 4

    Add app_mention — this is what triggers Adjutant when you @mention the bot.

  5. 5

    Click Save Changes.

Step 4 — Add Bot Scopes

  1. 1

    In the left sidebar, go to OAuth & Permissions.

  2. 2

    Scroll to Bot Token Scopes and add:

    • app_mentions:read — receive @mention events
    • chat:write — send messages
    • channels:join — auto-join the notification channel
    • channels:read — list channels
    • files:read — access shared files

Step 5 — Install the App to Your Workspace

  1. 1

    Still in OAuth & Permissions, scroll to the top and click Install to Workspace.

  2. 2

    Authorize the app.

  3. 3

    Copy the Bot User OAuth Token — it starts with xoxb-.

Step 6 — Invite the Bot to Directive Channels

  1. 1

    For any channel where you want to @mention the bot, type /invite @YourAppName in that channel.

  2. 2

    The notification channel is handled automatically — Adjutant will join it when you select it.

Step 7 — Connect in Adjutant

Bot Token (xoxb-...)→paste into the Bot Token field
App-Level Token (xapp-...)→paste into the App-Level Token field
Then: click Connect Slack, and once connected select your Notification Channel from the dropdown. @mention your bot in any channel to start sending directives.
